Simulation de gestion de mémoire virtuelle
===========================================

Projet L3.
L’objectif du projet est d’étudier les mécanismes à l’œuvre pour la gestion des pages mémoires.

Utilisation
===========

## Exécutable
    * make
    * ./myMemory -s tailleReelle -v tailleVirtuelle -algoChoisie
